Document Basics

Every HTML document has some basic text that needs to be there for it to work.
To start, you must have the text: <!DOCTYPE html> - you need this before any HTML so that the browser can recognize the document as an HTML document.
Next, you must have: <html> - this indicated that the following text is HTML.
If you want the browser tab to display a title, use <title>insert title here</title>
Afterward, you can have a header with <head> and </head>
To indicate the actual document, you will need to use <body> and </body>

Links and Images

What are links? Here's one.

To create a link, use the href attribute by doing <a href="insertlinkhere">linktext</a>


If you want to add images, it is easier to create an images folder on your site, to have local images. Then, use <img src="/images/insertimgfile"> After the image file, you can also change the length and the width by typing length="#" and width="#" before the closing tag, or >.

Lists and Tables

What are lists? Here's a small one.

  • Coffee
  • Milk
  • Tea

To create this unordered list, start with <ul>. Before the end of the element, you have to actually list the items that you want to list. For each item, use <li>listitemhere</li>. After all of your items, use </ul> to end your list.

Here's an example of a table.

First Name Last Name Points
Jill Smith 50
Eve Jackson 94

To begin the table, start with <table>. To create a row, use <tr>.Then, create the headings for the table. In this case you would type <th>First Name</th> <th>Last Name</th> and <th>Points</th>.
Next, you have to create another row. Then, fill the row with data, or <td>data</td>. In this case, it is <td>Jill</td> <td>Smith</td> and <td>50</td>
For the last row, create the last row and fill it with <td>Eve</td> <td>Jackson</td> and <td>94</td>
